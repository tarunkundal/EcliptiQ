## About
## EcliptiQ - Modern Team and Task Management Application

EcliptiQ is a dynamic and forward-looking team and task management application designed to revolutionize collaboration and elevate productivity. Skillfully developed using a fusion of cutting-edge technologies, including Vite, React.js, Chakra UI, TypeScript, Supabase and Redux Toolkit, EcliptiQ empowers teams to seamlessly organize tasks, foster effective communication, and effortlessly achieve their objectives.

## Features

- **User Authentication:** Secure user registration and login process with protected routes.
- **Team Creation:** Users can create and manage teams to streamline collaboration.
- **User Invitations:** Invite team members by email to join teams and collaborate.
- **Task Management:** Create, assign, and monitor tasks within your team.
- **Real-time Updates:** Experience real-time task updates and team communication.
- **State Management:** Utilizes Redux Toolkit for efficient state management.


## Technologies Used

- Vite: Lightning-fast build tool.
- React.js: Front-end library for building user interfaces.
- Chakra UI: Component library for modern UI design.
- TypeScript: A statically typed superset of JavaScript.
- Supabase: Backend-as-a-Service for authentication and data storage.
- Redux Toolkit: Efficient state management for complex applications.
